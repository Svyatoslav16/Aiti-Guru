# Общее описание

Приложения создано по шаблону vite
Используемые пакеты:
- MUI библиотека компонентов, также для реализации таблицы товаров
- Effector в качестве стейт менеджера
- zod для валидации схема
- react-hook-form для простой реализации формы авторизации, оверинжиниринг, но тут была задача показать, что есть опыт работы с таким
